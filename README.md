# Tic-Tac-Toe
Tic-Tac-Toe 2-player game in C:
- Users can learn game rules by pressing 'y/Y' and accessing a link.
- Options to start the game(Press 1) or view the leaderboard(Press 2).
- Players can enter their usernames and choose 'X' or 'O'.
- Moves are made using the numeric keypad.
- The first player to get three spaces in a row, column, or diagonal wins!
- Leaderboard shows the players who have played on the system.
- Users can continue the game by pressing '1' or end the game by pressing any other key.
